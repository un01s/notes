Here are some C code with the JavaScript quirks.
Credits to [luna](https://twitter.com/lunasorcery/status/1504893095661424644).

```C
#include <stdio.h>

int main() {
  puts("-0.5" + 1);
  printf("%d\n", 50 ** "2");
  return 0;
}
```

To build and run the above code, just do the following and ignore the warnings.

```Shell
$ gcc c-js.c && ./a.out
0.5
2500
```
