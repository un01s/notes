Nordic nRF5340

use the Toolchain Manager inside nRF Connect for Desktop to automatically install toolchains. After that use the command line to build a project. Current version is v2.1.1. $ cd /opt/nordic $ cd ncs/v2.1.1/nrf/applications/nrf5340_audio $ west build -b nrf5340_audio_dk_nrf5340_cpuapp --pristine -- -DCONFIG_AUDIO_DEV=1 -DCONF_FILE=prj_release.conf
